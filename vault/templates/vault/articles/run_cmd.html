command prompt cheat sheet

dir, ls, tree:

Type in dir or dir path_to_folder to view a list of directories. If you've installed git (properly), you'll be able to use cross-platform cmds such as ls instead of dir.

Tree is nice as well, try typing that in.

Interrupt Proces:

If a proces halts or becomes annoying, press ctrl+C to interrupt.

Wildcards:

If we want to select all files with a certain extension or name, you can use the wildcard * (eg: dir *.txt will show us all the files with the extension txt)

dir hello.* for file names and dir *.txt for file type

Note: dir *.* for EVERYTHING.

More tricks w. wildcards!

Filenames w. spaces:

In quotes, ie: del "my file"

Switches:

Help switch: /? (eg: help /?)

dir /p will print the dir one page at a time (press any key to proceed)

Switches work differently for different commands, so it is recommended to type in <command> /? to see what options you have.

Note: For a sorted list, type:

dir /b /s | sort

Running executable files:

Ex: type start notepad thisfile.txt to open the file in Notepad.

Any file that is an executable file can be run from the command line by typing the name of the file. For example, if you listed files using the dir command and see a file named "myfile.exe" typing "myfile" at the command line runs that program; typing in the suffix is not needed (you may, if you prefer, no problem :))

Change directory:

Typing cd .. goes up one level, and typing cd\ takes you to the C:\> prompt

Copying files:

You can use the copy command from the command line to copy files.

copy c:\myfile.txt d:

copy *.txt e:

The above command uses a wildcard to copy all text files in the current directory to the E: drive.

		copy f:\example.xls

The above command would copy the file "example.xls" on the F:\ drive to the current directory. Notice that we did not specify a destination; if the destination is not specified, the current directory is used by default.

Xcopy copy method (advanced):

Using the xcopy command, you can copy the all the files from one drive to another drive. Example:

		xcopy /h /c /k /e /r /y c:\ d:\

The above command copies all of the files on the C:\ drive to the D:\ drive. There are many options specified here (the letters with a slash before them). Here's what they do:

Option

Meaning

/h

Copy hidden and system files. Normally xcopy skips these files, but if you specify this option, they will be copied.

/c

Continue copying even if an error is encountered. This can be helpful if you need to step away from the computer, and you know that you won't need to stop the copy operation, even if there are errors.

/k

Keep read-only attribute. If you specify this option, read-only files will retain the read-only file attribute when they are copied.

/e

Copy empty directories. Normally, empty directories are not copied. If this option is specified, xcopy will copy all directories, even if they are empty.

/r

Overwrite read-only files in destination. Normally, if read-only files exist in your destination which would be overwritten by the copy, xcopy will stop and tell you "Access denied." If this option is specified, it will overwrite them and continue copying.

/y

Assume yes to all overwrites. Normally, if a file would be overwritten by the copy, xcopy will ask you to confirm before it overwrites. If this option is specified, it will overwrite them without asking you.

Create files:

Create a .txt file:

start notepad myfile.txt

Batch files:

Start a batch file:

start notepad example.bat

Insert the following:

@echo off
cls
dir

Save the file (normally; using the GUI) and run it by going to it's directory and typing in example

Moving a file:

Assuming example.bat exists:

mkdir dir2

move example.bat dir2

Rename a file:

rename example.bat first.bat

Deleting files:

del hello.txt

Tip: When deleting files you can also use wildcards to delete multiple files at once. For example, if the directory contained several .GIF image files you could type del *.gif to delete all files ending with the .gif file extension.

Rename a file/directory:

rename dir2 "my special folder"

Remove directory:

rmdir "my special folder"

Note: In case the folder contains files, rmdir /s "my special folder" to destroy the files along with the folder.

Pinging:

ping google.com

ping 167.128.0.13

List ping path:

pathping <IP or domain address>

Trace route:

tracert facebook.com

tracert 192.145.0.2

Retrieve meta:

ipconfig /all

See also:

ipconfig /flushdns
ipconfig /release
ipconfig /renew

Print ipconfig to a .txt file:

ipconfig /all > network_info.txt

dir > me_dirextareee.txt

Note: will print to current working directory (cwd)

To append data to an existing file, use: >>

To "go backwards", snatching data from the .txt file and inserting it into an (empty) cmd prompt, simply reverse the handle:

sort < network_info.txt

Formatting drive:

format c: /q /u

Create secure folder:

browse to desired location and:

md aux\

this folder can neither be moved nor deleted.

To get rid of it:

rd aux\

Create Hidden Folder:

mkdir secrets

attrib +h +s +r secrets

Secrets is, of course, an arbitrary name; name it as you please.

This will even circumvent "show hidden files and folders".

Note: name it something that you can remember; you'll need to in order to bring it back:

attrib -h -s -r secrets

Clear Screen:

cls

Shutdown w. cmd prompt:

shutdown /s

shutdown /r

shutdown /l

shutdown, reboot & logoff, respectively

To set a timer for shutdown in 50 seconds w. a message:

shutdown /s /t 50 /c "I'm shutting down now, buddy!"

Create WiFi hotspot:

Set up:

netsh wlan set hostednetwork mode=allow ssid=<insert host here> key=<insert login here (min 8 chars)>

Start:

netsh wlan start hostednetwork

Stop it again:

netsh wlan stop hostednetwork

View all installed programmes:

wmic product get name

Network statistics:

netstat ?/

Associated items:

View complete list of associations between file types and the programs that run them.

assoc

To set associations insert syntax as seen below:

assoc .txt=word

Wipe file content:

When you delete files in Windows, what you are actually doing is destroying the file path. The data remains and can be retrieved. Cipher puts an end to that nonsense.

cipher

View installed drivers:

driverquery -v

view currrently open ports and their states:

netstat -an

Retrieve System Info:

For a list of installed hardware:

systeminfo

Power Configuration:

view available energy states:

powercfg -a
view energy consumption report:

powercfg /energy

Also see:
powercfg /hibernate on

powercfg /hibernate off

Sfc - view and repair win sys files:

sfc /scannow

Tasklist:

advanced task manager, basically

tasklist

show services:

tasklist -svc

more details:

tasklist -v

Kill it with fire:

taskkill -im <exe name>

taskkill -pid <exe name>

Arp:

arp: - adress resolution protocol; used to view and manage the ARP cache
- resolves MAC- and IP-adresses

arp # cmd looks up docstring on ARP
arp -a # lists all recent MAC and IP-adresses
arp -d <IP adress> # delete an IP in the ARP

Clip command:

Send the output returned by any cmd to the clipboard thusly (we'll use ipconfig as an example):

ipconfig | clip
